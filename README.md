# JS Games & Mini-Projects 🚀

A collection of interactive JavaScript applications and games. This repository is dedicated to mastering core web fundamentals—including DOM manipulation, state management, browser APIs, and algorithmic game logic—to build a rock-solid foundation for complex MERN stack architecture.

## 📌 Projects Included

### 1. Advanced Todo App (State Management)
A robust task tracker built to mimic modern frontend framework concepts without actually using a framework.
* **Key Concepts:** Single Source of Truth (State array), SessionStorage for data persistence, Event Delegation for optimized performance, and full CRUD (Create, Read, Update, Delete) operations.

### 2. Interactive Tier List (Drag & Drop)
A dynamic ranking board where users can drag and drop images into color-coded tiers. 
* **Key Concepts:** HTML5 Drag and Drop API, dynamic DOM node re-parenting, array shuffling algorithms, and responsive CSS Flexbox/Grid layouts.

### 3. Ping Pong Game
A smooth, retro arcade tennis game featuring a custom physics and game loop.
* **Key Concepts:** Delta-time scaling (`requestAnimationFrame`) for consistent speed across different monitor refresh rates, AABB (Axis-Aligned Bounding Box) collision detection, and asynchronous key-state tracking for stutter-free movement.

### 4. Classic Snake Game
The classic Nokia-style snake game built on a coordinate grid system.
* **Key Concepts:** Matrix coordinate math, matrix boundaries, interval loop management, and array manipulation (`unshift` and `pop`) to simulate movement and growth.

### 5. Simon Memory Game
A sequence-memorization game that progressively gets faster and harder.
* **Key Concepts:** Asynchronous JavaScript (`async/await` and Promises) to replace messy `setInterval` loops, sequential timing logic, and advanced CSS pseudo-elements (`::after`) for glowing visual feedback.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and Browser APIs (Drag and Drop).
* **CSS3:** Custom variables, Flexbox, CSS Grid, and keyframe animations.
* **JavaScript (ES6+):** Arrow functions, destructuring, Promises, Local/Session storage, and dynamic DOM manipulation.
